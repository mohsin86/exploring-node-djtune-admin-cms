# exploring-node-djtune-admin-cms

a simple architectural CMS with node.js, moongoose, express, handalbars 

## Technologies
Following module is used for building this project :-
* express
* express-handlebars
* express-validator
* bcrypt
* express-session
* body-parser

~~~~
## Setup
To run this project, install it locally using npm:

$ cd ../lorem
$ npm install
$ npm start
~~~~
